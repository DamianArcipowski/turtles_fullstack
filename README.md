1. Należy się upewnić, że w systemie zainstalowany jest Docker engine wraz z pluginem Docker Compose (w systemie Windows Docker Desktop)
2. W katalogu aplikacji "docker" znajduje się plik "docker-compose.yaml"
3. Należy wejść do powyższego katalogu i uruchomić polecenie "docker compose up"
4. Następnie odczekać chwilę aby kontenery się zbudowały i wystartowały
5. Aplikacja wystartuje na "http://localhost:8080"
